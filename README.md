This repository contains a simple backend database system for an online store order management application implemented in PostgreSQL. 

The project includes database schema initialization, PL/pgSQL functions for calculations, automated stored procedures for order creation, and triggers for real-time order total recalculations and audit logging.

 `solution.sql` — Main script containing the schema, functions, procedures, triggers, and test cases.
 `answers.md` — Theory questions regarding functions, procedures, and triggers.
